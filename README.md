# Marketing Calculator

Простой калькулятор метрик для рекламных кампаний в стиле macOS.

## Использование

Откройте [index.html](index.html) в браузере или разверните на GitHub Pages.

## Формулы

- **Daily Clean Profit** = Чистая прибыль в месяц / 30
- **ROI в десятичных** = ROI целевой (%) / 100
- **Daily spend** = Daily Clean Profit / ROI
- **Revenue** = Daily spend + Daily Clean Profit
- **Sale** = Sale (продаж) / (ROI + 1)
- **Lead** = Sale / 4
- **CR(Lead)** = CPC / Lead
- **CR(sale)** = 1 / (Sale / CPC)
- **CPL в Gads** = Lead - (Lead × Коэффициент CPL)
